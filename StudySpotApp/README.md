# Study Spot App

## About

The Study Spot App helps students find quiet places to study.

Users will be able to filter locations by:

- Quietness
- Coffee Shops
- Libraries
- Nice Views
- Low Foot Traffic
- Wi-Fi
- Power Outlets
- Open 24/7
